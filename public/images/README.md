# Banner Ad Poster Images

This directory should contain poster images for the banner ad videos.

## Required Poster Images

### consumerism-poster.jpg
- **Purpose**: Poster/thumbnail for the consumerism video
- **Recommended specs**: 
  - Resolution: 320x480px (portrait orientation)
  - Format: JPG or PNG
  - File size: Keep under 200KB
- **Content**: Should represent the video content well - shopping, consumer goods, retail scenes

### subscriptions-poster.jpg
- **Purpose**: Poster/thumbnail for the subscriptions video
- **Recommended specs**: 
  - Resolution: 320x480px (portrait orientation)
  - Format: JPG or PNG
  - File size: Keep under 200KB
- **Content**: Should represent the video content well - subscription services, digital lifestyle

## Usage

These poster images are displayed while the video is loading and provide a fallback if the video fails to load. They should be visually consistent with the video content and work well with the CRT effects applied to the banner ads.

## Adding Posters

1. Create poster images that match the video content
2. Place them in this directory with the exact names specified
3. Ensure they are optimized for web (compressed, appropriate file size)
4. Test to ensure they display correctly in the banner ads
