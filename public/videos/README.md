# Banner Ad Videos

This directory should contain video files for the banner ads.

## Required Videos

### consumerism.mp4
- **Purpose**: Background video for the "overconsumption" banner ad
- **Recommended specs**: 
  - Resolution: 320x480px (portrait orientation for banner)
  - Duration: 10-30 seconds (will loop)
  - Format: MP4 with H.264 codec
  - File size: Keep under 5MB for fast loading
- **Content suggestions**: 
  - Shopping scenes, consumer goods, retail environments
  - Fast-paced, colorful, engaging visuals
  - Should work well with CRT effects

### subscriptions.mp4
- **Purpose**: Background video for the "subscriptions" banner ad
- **Recommended specs**: 
  - Resolution: 320x480px (portrait orientation for banner)
  - Duration: 10-30 seconds (will loop)
  - Format: MP4 with H.264 codec
  - File size: Keep under 5MB for fast loading
- **Content suggestions**: 
  - Subscription services, streaming, digital services
  - Modern tech lifestyle, convenience themes
  - Should work well with CRT effects

## CRT Effect Compatibility

The videos will automatically have CRT effects applied:
- Scanlines overlay
- Vignette effect
- Contrast and saturation adjustments
- Flicker animation

Choose videos with good contrast and avoid very dark or very bright content that might not work well with the CRT overlay effects.

## Adding Videos

1. Place your video files in this directory
2. Ensure they are named exactly as specified above
3. Test the banner ads to ensure the videos load and play correctly
4. Consider creating poster images in `/public/images/` for better loading experience
